Help using the prototype

After clicking 'start now', there are 2 options depending on which user journey you want to explore. Use the:

  • 'find all courses nearby' option for the browse journey
  • 'answer a few questions for more specific results' option for the search guided journey

In the guided journey, enter 'business' into the 'jobs or careers' question to see business-related results. The first 2 results on the search results screen can be viewed.

You can also enter 'biology' to see biology and science-related results.

Go directly to the screens showing an example of when there are few search results and an example of when there are no search results.

Note that most external links, including those to provider websites, are not currently working.
